# Bharat Darshan

## Project Description
Bharat Darshan is a web-based application developed to present information about India’s culture, heritage, travel destinations, and blogs. The project includes a dynamic blog module integrated with a backend system to store and retrieve blog data along with images.

The project demonstrates frontend and backend integration using standard web technologies.

---

## Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript
- Bootstrap 4
- Font Awesome

### Backend
- Node.js
- Express.js

### Database
- SQLite

### Libraries
- Multer (for image upload handling)
